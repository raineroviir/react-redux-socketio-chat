# React-redux-socketio-chat

### This project is a work-in progress.  



####
Use Guide:
```
- npm install
- grunt build
- npm start
```

#####
if you want to setup mongo:

```
- mkdir db
- mongod --dbpath=./db --smallfiles
```

#####
To see the live version of the app go to http://slackclone.herokuapp.com
